## 对象创建
```
let obj0 = {
    name:"vhxn",
    age:18,
    address:{
        city:"shenzhen",
    }
}

let obj1 = {
    name:"vhxn1",
    age:19,
    address:{
        city:"shenzhen1"
    }
}
```

## 打印对象
```
console.log(obj0)
console.log(obj1)
```
